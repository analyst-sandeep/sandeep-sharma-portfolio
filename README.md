# Sandeep Sharma Portfolio Site

Static portfolio website for Sandeep Sharma, Data Analyst.

## Includes

- Updated CV download
- Work experience summary
- Retail Customer Segmentation & Revenue Analytics project
- Customer Churn & Retention Analytics project
- Skills and contact section

## Run Locally

Open `index.html` directly in a browser, or serve the folder with:

```bash
python -m http.server 8020
```
